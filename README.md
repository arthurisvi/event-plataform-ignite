# Event Plataform 
Projeto desenvolvido durante o Ignite Lab-02 da Rocketseat.
Na aplicação foi construída uma página de uma plataforma que simula a exibição de vídeo aulas de um curso. 

### Tecnologias utilizadas
- ReactJS + TypeScript
- GraphQL
- GraphCMS
- Apollo Client
- TaiwilndCSS
- Vitejs
- React Router
